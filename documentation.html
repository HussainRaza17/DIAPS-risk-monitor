Getting Started

Welcome to the DIAPS API! Our REST API provides access to real-time disaster monitoring data, weather information, earthquake tracking, and AI-powered risk assessments.

🔑 Get Your API Key

To use the DIAPS API, you'll need an API key. Enter your email below to generate one:

Your API Key:


                  
                

              

            


            
Base URL

            

              
https://api.diaps.org/v1

              
            


            
Quick Start Example

            

              
curl -H "X-API-Key: YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     "https://api.diaps.org/v1/weather/current?lat=35.6762&lng=139.6503"

              
            

          


          

            
Authentication

            
All API requests require authentication using an API key. Include your API key in the request header:


            

              
X-API-Key: YOUR_API_KEY

              
            


            

              
              

                Security: Keep your API key secure and never expose it in client-side code. Use
                environment variables or secure key management services.

Weather API

            
Access real-time weather data and forecasts for any location worldwide.


            

              

                GET
                /weather/current
              

              
Get current weather conditions for a specific location.


              
Parameters

              

                
                  

                    Parameter
                    Type
                    Required
                    Description
                  

                
                
                  

                    lat
                    float
                    Yes

                    lng
                    float
                    Yes
                    Longitude (-180 to 180)
                  

                  

                    units
                    string
                    No
                    metric, imperial (default: metric)

Earthquake API

            
Access real-time earthquake data and seismic activity monitoring.


            

              

                GET
                /earthquakes/recent
              

              
Get recent earthquake data within a specified radius and time period.


              

                
                  

                    Parameter
                    Type
                    Required
                    Description
                  

                
                
                  

                    lat
                    float
                    Yes
                    Latitude (-90 to 90)
                  

                  

                    lng
                    float
                    Yes
                    Longitude (-180 to 180)
                  

                  

                    radius
                    integer
                    No
                    Search radius in km (default: 100)
                  

                  

                    min_magnitude
                    float
                    No
                    Minimum magnitude (default: 2.0)
                  

                  

                    days
                    integer
                    No
                    Days to look back (default: 7)
                  

                
              


              
Example Response (200 OK)

Risk Assessment API

            
Get AI-powered disaster risk assessments and predictions for any location.


            

              

                POST
                /risk/assess
              

              
Generate comprehensive risk assessment for natural disasters.


              
Request Body

  "location": {
    "lat": 35.6762,
    "lng": 139.6503
  },
  "assessment_types": ["earthquake", "weather", "flood", "wildfire"],
  "time_horizon": "30_days"
}

Alerts API

            
Manage disaster alerts and notifications for your applications.


            

              

                GET
                /alerts/active
              

              
Get active disaster alerts for a specific region.


              

                
                  

                    Parameter
                    Type
                    Required
                    Description
                  

                
                
                  

                    lat
                    float
                    Yes
                    Latitude (-90 to 90)
                  

                  

                    lng
                    float
                    Yes
                    Longitude (-180 to 180)
                  

                  

                    radius
                    integer
                    No
                    Search radius in km (default: 50)
                  

                  

                    severity
                    string
                    No
                    low, moderate, high, critical
                  

                
              

            


            

              

                POST
                /alerts/subscribe
              

              
Subscribe to disaster alerts via webhook or email.

            

          


          

            
Locations API

            
Search and manage location data for disaster monitoring.


            

              

                GET
                /locations/search
              

              
Search for locations by name or coordinates.


              

                
                  

                    Parameter
                    Type
                    Required
                    Description
                  

                
                
                  

                    q
                    string
                    Yes
                    Search query (city, address, etc.)
                  

                  

                    limit
                    integer
                    No
                    Max results (1-20, default: 10)
                  

                
              

            

          


          

            
Error Handling

            
The API uses standard HTTP status codes and returns detailed error information.


            
Common Error Codes

            

              
                

                  Code
                  Status
                  Description
                

              
              
                

                  400
                  Bad Request
                  Invalid request parameters
                

                

                  401
                  Unauthorized
                  Invalid or missing API key
                

                

                  429
                  Too Many Requests
                  Rate limit exceeded
                

                

                  500
                  Internal Server Error
                  Server error occurred
                

              
            


            
Error Response Format

            

              

                400 Bad Request
              

              

                
{
  "error": {
    "code": "INVALID_PARAMETERS",
    "message": "Invalid latitude value. Must be between -90 and 90.",
    "details": {
      "parameter": "lat",
      "value": "invalid_value",
      "expected": "float between -90 and 90"
    }
  },
  "request_id": "req_abc123",
  "timestamp": "2025-06-13T10:30:00Z"
}

                
              

            

          


          

            
SDKs & Libraries

            
Official SDKs and community libraries to help you integrate DIAPS API quickly.


            

              
🐍 Python SDK

              

                
pip install diaps-python

                
              

              

                
from diaps import DiapsClient

client = DiapsClient(api_key="YOUR_API_KEY")
weather = client.weather.current(lat=35.6762, lng=139.6503)
print(weather.temperature)

                
              

            


            

              
🟨 Node.js SDK

              

                
npm install diaps-node

                
              

              

                
const { DiapsClient } = require('diaps-node');

const client = new DiapsClient({ apiKey: 'YOUR_API_KEY' });
const weather = await client.weather.current({ lat: 35.6762, lng: 139.6503 });
console.log(weather.temperature);

                
              

            


            

              
☕ Java SDK

              

                
<dependency>
    <groupId>org.diaps</groupId>
    <artifactId>diaps-java</artifactId>
    <version>1.0.0</version>
</dependency>

                
              

              

                
DiapsClient client = new DiapsClient("YOUR_API_KEY");
Weather weather = client.weather().current(35.6762, 139.6503);
System.out.println(weather.getTemperature());
